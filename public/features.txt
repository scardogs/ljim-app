4. API ENDPOINTS
================================================================================

4.1 AUTHENTICATION ENDPOINTS
----------------------------
POST   /api/auth/login          - Admin login
POST   /api/auth/register       - Admin registration
POST   /api/auth/verify         - Token verification

4.2 ADMIN CONTENT ENDPOINTS
---------------------------
GET    /api/admin/homepage      - Get homepage content
PUT    /api/admin/homepage      - Update homepage content

GET    /api/admin/about         - Get about page content
PUT    /api/admin/about         - Update about page content

GET    /api/admin/events        - Get all events
POST   /api/admin/events        - Create new event
PUT    /api/admin/events        - Update event
DELETE /api/admin/events        - Delete event

GET    /api/admin/give          - Get give page content
PUT    /api/admin/give          - Update give page content

GET    /api/admin/shop          - Get shop content
PUT    /api/admin/shop          - Update shop content

GET    /api/admin/contact       - Get contact page content
PUT    /api/admin/contact       - Update contact page content

GET    /api/navbar/content      - Get navbar content
PUT    /api/navbar/content      - Update navbar content (admin)

4.3 PRAYER REQUEST ENDPOINTS
----------------------------
GET    /api/prayer-requests     - Get approved public requests
POST   /api/prayer-requests     - Submit new prayer request

GET    /api/prayer-requests/content         - Get page content
PUT    /api/prayer-requests/content         - Update page content (admin)

GET    /api/admin/prayer-requests           - Get all requests (admin)
GET    /api/admin/prayer-requests/[id]      - Get single request (admin)
POST   /api/admin/prayer-requests/[id]/approve   - Approve request
POST   /api/admin/prayer-requests/[id]/unapprove - Unapprove request
DELETE /api/admin/prayer-requests/[id]      - Delete request

POST   /api/prayer-requests/[id]/pray       - Increment prayer count

4.4 MUSIC ENDPOINTS
-------------------
GET    /api/songs               - Get all songs
POST   /api/songs               - Create new song (admin)
GET    /api/songs/[id]          - Get single song
PUT    /api/songs/[id]          - Update song (admin)
DELETE /api/songs/[id]          - Delete song (admin)

GET    /api/singers             - Get all singers
POST   /api/singers             - Create new singer (admin)
GET    /api/singers/[id]        - Get single singer
PUT    /api/singers/[id]        - Update singer (admin)
DELETE /api/singers/[id]        - Delete singer (admin)

4.5 USER MANAGEMENT ENDPOINTS
-----------------------------
GET    /api/admin/users         - Get all admin users
POST   /api/admin/users         - Create new admin user
PUT    /api/admin/users         - Update admin user
DELETE /api/admin/users         - Delete admin user

4.6 IMAGE MANAGEMENT ENDPOINTS
------------------------------
POST   /api/admin/upload-image  - Upload new image
DELETE /api/admin/delete-image  - Delete image

4.7 CHAT ENDPOINTS
------------------
POST   /api/chat/gemini         - Send chat message to AI
POST   /api/chat/backup         - Backup chat endpoint
