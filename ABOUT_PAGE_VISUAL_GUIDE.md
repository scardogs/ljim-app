## ✨ Interactive Elements

### Hover Effects:

```
Info Cards (Before Hover):
┌──────────────────┐
│  📖              │
│                  │
│  Our Story       │
│                  │
│  Content...      │
│                  │
└──────────────────┘

Info Cards (On Hover):
    ┌──────────────────┐
    │  📖  ✨         │  ← Lifts 8px up
    │                  │
    │  Our Story       │  ← Bigger shadow
    │                  │
    │  Content...      │
    │                  │
    └──────────────────┘
```

### Animations:

1. **Page Load:**

   ```
   Opacity: 0 → 1
   Position: y+20px → y+0px
   Duration: 0.6s
   Stagger: 0.1s delay between cards
   ```

2. **Main Heading:**

   ```
   Shimmer effect:
   ←─────────────────→
   6 second cycle
   Infinite loop
   ```

3. **Background:**
   ```
   Gradient shift:
   0% → 100% → 0%
   30 second cycle
   Infinite smooth loop
   ```
